## 张量操作总结

#### 1. **标量**
- 标量由只有一个元素的张量表示。
- 示例：
  ```python
  x = torch.tensor([3.0])
  y = torch.tensor([2.0])
  x + y, x * y, x / y, x ** y
  ```
#### 2. **向量**
- 向量可以看作标量值的列表。
- 示例：
  ```python
  x = torch.arange(4)
  x[3]  # 访问向量的元素
  len(x)  # 向量的长度
  x.shape  # 向量的形状
  ```

#### 3. **矩阵**
- 创建矩阵：
  ```python
  A = torch.arange(20).reshape(5, 4)
  ```
- 矩阵的转置：
  ```python
  A.T
  ```
- 对称矩阵：
  ```python
  B = torch.tensor([[1, 2, 3], [2, 0, 4], [3, 4, 5]])
  B == B.T
  ```

#### 4. **高维张量**
- 创建具有更多轴的张量：
  ```python
  X = torch.arange(24).reshape(2, 3, 4)
  ```

#### 5. **按元素操作**
- 张量的按元素加法：
  ```python
  A = torch.arange(20, dtype=torch.float32).reshape(5, 4)
  B = A.clone()
  A + B
  ```
- 哈达玛积（按元素乘法）：
  ```python
  A * B
  ```

#### 6. **广播机制**
- 标量与张量的操作：
  ```python
  a = 2
  X = torch.arange(24).reshape(2, 3, 4)
  a + X, (a * X).shape
  ```

#### 7. **求和与均值**
- 元素求和：
  ```python
  x = torch.arange(4, dtype=torch.float32)
  x.sum()
  ```
- 按轴求和：
  ```python
  A.sum(axis=0)
  A.sum(axis=[0, 1])
  ```
- 均值：
  ```python
  A.mean(), A.sum() / A.numel()
  ```

#### 8. **保持轴数不变**
- 求和时保持轴数不变：
  ```python
  sum_A = A.sum(axis=1, keepdim=True)
  A / sum_A
  ```

#### 9. **累积操作**
- 累积求和：
  ```python
  A.cumsum(axis=0)
  ```

#### 10. **点积**
- 向量点积：
  ```python
  torch.dot(x, y)
  torch.sum(x * y)
  ```

#### 11. **矩阵运算**
- 矩阵-向量乘法：
  ```python
  torch.mv(A, x)
  ```
- 矩阵-矩阵乘法：
  ```python
  B = torch.ones(4, 3)
  torch.mm(A, B)
  ```

#### 12. **范数**
- L2范数（向量元素平方和的平方根）：
  ```python
  u = torch.tensor([3.0, -4.0])
  torch.norm(u)
  ```
- L1范数（向量元素绝对值的和）：
  ```python
  torch.abs(u).sum()
  ```
- 矩阵的弗罗贝尼乌斯范数：
  ```python
  torch.norm(torch.ones((4, 9)))
  ```

## 思考
### 易混淆点
1. torch.mv 是矩阵（m×n）-向量的乘积（向量是张量为1的，长度为n，会自动调整为m×1）
2. torch.mm 是矩阵-矩阵的乘积（m×a矩阵要对应a×n矩阵，第一个列等于第二个行）
