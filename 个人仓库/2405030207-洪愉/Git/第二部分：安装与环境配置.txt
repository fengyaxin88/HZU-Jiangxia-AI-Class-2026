一、核心内容
1.多平台安装：
Linux：通过系统包管理器安装，如 sudo apt install git(Debian/Ubuntu)。
macOS：可通过 Homebrew (brew install git) 或安装 Xcode 命令行工具获取。
Windows：从官网下载安装程序，安装后会提供 Git Bash​ 命令行工具。
2.必需的首步配置：安装后必须立即设置全局用户信息，此信息会关联到你的每一次提交。
git config --global user.name “你的姓名”
git config --global user.email “你的邮箱”
3.验证与检查：
使用 git --version验证安装。
使用 git config --global --list查看已生效的全局配置。

二、思考
在分布式的协作里，配置邮箱和姓名很重要，全靠这个信息来追溯每一条修改是谁做的，这是协同工作的信任基础。用命令行操作，更直接高效，对理解Git底层的工作有很大的帮助。

三、demo
# 1. 检查是否安装成功及版本号
git --version
# 2. 设置全局身份标识
git config --global user.name “张三”
git config --global user.email “zhangsan@example.com”
# 3. 查看全局配置，确认信息已设置
git config --global --list
